【重要】使用 Web3Modal 版本（推荐）
=====================================

新版本使用专业的钱包连接插件 Web3Modal，支持更多钱包，更稳定！

快速开始（3 步）：
------------------

步骤 1：获取 Project ID
  1. 访问：https://cloud.walletconnect.com/
  2. 注册/登录账号
  3. 创建新项目
  4. 复制 Project ID

步骤 2：配置代码
  1. 打开 app-web3modal.js
  2. 找到第 6 行：const projectId = 'YOUR_PROJECT_ID'
  3. 替换 YOUR_PROJECT_ID 为您的 Project ID
  4. 保存文件

步骤 3：启动服务器（重要！）
  ⚠️ 不要直接双击 HTML 文件，会出现 404 错误！
  
  方法一（推荐）：
    - 双击 start-server.bat
    - 浏览器会自动打开
  
  方法二：
    - 打开命令行
    - 进入项目目录
    - 执行：python -m http.server 8000
  
  然后访问：http://localhost:8000/index-web3modal.html
  
  ✓ 正确：地址栏显示 http://localhost:8000/...
  ✗ 错误：地址栏显示 file:///D:/...

主要功能：
---------
1. 🔐 自动加密存储密码（AES-256-GCM）
2. 🔓 自动解密查看密码
3. 👁️ 密码显示/隐藏切换
4. 📋 一键复制密码到剪贴板
5. 📝 编辑密码信息
6. 🗑️ 删除密码
7. 🔍 搜索和分类筛选
8. 📱 移动端支持（WalletConnect）

🔒 安全特性：
- 端到端加密（钱包签名生成密钥）
- 只有您能解密自己的密码
- 密码在上链前加密，下链后解密
- 密钥不存储，每次通过签名重新生成

常见问题：
---------
Q: 出现 404 错误？
A: ❌ 不要直接双击 HTML 文件！
   ✓ 必须运行 start-server.bat 启动服务器
   ✓ 通过 http://localhost:8000 访问
   详见：404错误修复指南.md

Q: 钱包无法连接？
A: 确保已正确配置 Project ID，并通过 HTTP 服务器访问

Q: 页面打不开？
A: 运行 troubleshoot.bat 进行诊断

Q: 网络错误？
A: 按照钱包提示切换到 XLayer 网络（Chain ID: 196）

Q: 端口 8000 被占用？
A: 使用其他端口：python -m http.server 8080
   然后访问：http://localhost:8080/index-web3modal.html
